# How To Use

Must be a cover.jpg

Stripped out local photos. Can add - /path/to/photos:/data/photos and remove GCS_BASE_URL to work locally

build.php can help make thumbs/display files of the right size

If there are a lot of images then takes a long time to render - always be the
first to load it! Or schedule a job?

Files are in alphabetical order - not order on disk

Use build.php to create thumbnails needed - suggest run locally then upload to the storage layer if using GCS like I am
default to share them - need public links (API just not worth the protection)
